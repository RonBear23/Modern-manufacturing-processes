Project 2

The assignment

Project 2 was split in three parts, a group exercise, where we measure kerf, an individual project where we had to create a parametric model of an object to cut and assemble it with pressfit box joints, and finally a we make a sticker using a vinyl cutter.

Kerf

My friend and teammate for this exersice, Benni, is going to take you through how we went on measuring kerf on his website here

Pressfit lasercut project

I really don't like working and studying at home. One of the reason for that is that my desk is messy and crowded with stuff. Messy desk makes it more difficult for me to concentrate

I have 4 file folders, stuffed with my alot of uneccesary stuff f.x. a history book on ancient greece from high school, and it takes up to much space. So my idea is to make one neat little file folder only for important stuff and a space for me to put pens and stuff. That way I get rid of all the unimportant files and books and create more space on my desk. Less is sometimes more.

I started by making some sketches to make it easier for when I start drawing on a computer.

Now I was almost ready to start drawing but I first had to learn how to make it parametric and press-fit. I watched this cool video to help me. Here is how I made it parametric

I was now ready to start drawing. I used Fusion 360. Here's how it went

In fusion you can put in the kerf that you want and it will adjust. I put it at 0.18mm. Then, export the file as a dxf file and open it up in Inkscape. Inkscape is vector graphics edidtor to create vector images. It is now ready for laser cutting. I used plywood, 4mm thickness, to cut out of. This is the final product.

The press-fit fitted nicely the only problem was I didn't add more. In the instructions video that I followed the gentleman only put fittings on the bottom but not for the sides. I should've done that and not follow the video so precisely. But it does its job and I am happy with the result.

Making a sticker using a vinyl cutter

The final project of this assignment was to make a sticker using a vinyl cutter. I chose to make a Butters, from South Park, sticker. I see many similarities between us. I found a black and white svg(scalable graphics vector) figure of Butters. Import into Inkscape and trace the figure, make adjustments and export as svg.

Love it.
